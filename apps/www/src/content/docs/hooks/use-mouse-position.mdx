Tracks the mouse position return its X and Y.

### Installation
<Code code="npx rabbithook@latest add use-mouse-position" lang="bash" frame={false} />

### Usage
<Code
    lang="tsx"
    title="App.tsx"
    frame="code"
    code={`
import useMousePosition from "@/hooks/use-mouse-position";

function Component() {
  const { x, y } = useMousePosition();

  return (
    <p>Mouse position: {x}, {y}</p>
  );
}
`}
  />
