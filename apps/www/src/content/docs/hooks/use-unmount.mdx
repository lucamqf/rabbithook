Just like useEffect but runs only when the component unmounts.

### Installation
<Code code="npx rabbithook@latest add use-unmount" lang="bash" frame={false} />

### Usage
<Code
    lang="tsx"
    title="App.tsx"
    frame="code"
    code={`
import useUnmount from "@/hooks/use-unmount";

function Component() {
  useUnmount(() => {
    console.log("Component unmounted");
  })

  return ();
}
`}
  />
